2018 · 起點

沒人教、沒資源，
所以我自己學、自己做。

在視宇擔任業務期間，為了讓自己更有說服力，自費成為 Amazon + eBay 賣家——自己找供應商、建產品、搞定物流金流，用真實的戰場學習跨境電商。

這一年在學的事
🛒
平台入門
從開設賣家帳號開始，摸索 Amazon Seller Central 和 eBay 後台的每一個功能，理解平台規則和演算法邏輯。熟悉從帳號申請、商品上架到後台數據分析的完整流程。
📦
跨境物流
了解國際運輸成本結構、FBA 倉儲邏輯、報關文件，把物流成本納入選品評估。從台灣出貨到美國倉儲的整條供應鏈，一手建立並持續優化。
✍️
Listing 優化
學習英文標題、bullet points、A+ content 的寫法，讓商品在搜尋結果中脫穎而出。透過關鍵字研究和競品分析，持續提升商品曝光率與轉換率。
核心技能
Amazon Seller Central eBay 賣家後台 FBA 倉儲 跨境物流 選品研究 Listing 優化 競品分析
AI 工具能力
Claude Manus Gemini Python GitHub
反思

公司沒有電商資源，所以我自己出錢出時間去當賣家。

這不只是學習，而是用真實的虧損和獲利逼自己搞懂每一個環節。

後來跟客戶談，說的每一句話都是親身經歷——這是任何課程給不了的說服力。
